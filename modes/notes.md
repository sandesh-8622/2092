# Mode: notes

Save notes to `data/notes/` as markdown files. Name them descriptively.

If they say "save this as a note" or "add to my [topic] notes," do it. If a file for that topic exists, append. If not, create it.

They should be able to say "show me my notes on [topic]" and you pull up the right file. Or "what notes do I have" and you list everything in the folder.

Keep notes clean and organized. Use headers, dates, and context.
